# Controle-de-Estoque---ltima-vers-o
Sistema de gestão de estoque, com diversas funcionalidades de compra e venda de produtos e movimentações financeiras

Novidades nesta versão:
- pedido de compras e de vendas - possibilidade de fazer a compra e a venda de vários produtos na mesma operação - com limites de saldo para compras e limite de estoque para vendas
- Limites de estoque - referenciais de quando o estoque está baixo, está aceitável ou está alto
- Produtos em baixa - revela os produtos cujo estoque está baixo, para facilitar a percepção de quais produtos de deve adquirir para facilitar novas negociações
- Reestruturação do código para ficar menos repetitivo e mais eficiente
- Cálculos automáticos durante o pedido de compras e de vendas para facilitar a previsão da operação
